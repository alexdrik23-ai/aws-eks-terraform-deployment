# AWS EKS Deployment with Terraform

## Overview

This project provisions an Amazon EKS cluster using Terraform and is designed to deploy an application.

## Skills Demonstrated

- AWS EKS
- Terraform
- Kubernetes
- VPC and networking basics
- Infrastructure as Code

## Project Goal

The goal of this project is to build a reusable EKS environment and prepare it for Kubernetes application deployments.

## Repository Structure

- `provider.tf` - AWS provider configuration
- `main.tf` - Main infrastructure resources
- `variables.tf` - Input variables
- `outputs.tf` - Output values
- `terraform.tfvars.example` - Example variable values
